后台获取的参数必须去除首尾空白符后，再进行逻辑运算

特殊字符必须原样存库，取的时候进行转义

数据库所有列必须有默认值，不能为 null，不能存 '' 空串以外的空白串

所有函数都需要做入参校验，每一个参数都要校验

Java 中，有返回值的函数，只有正常执行才有返回结果，否则都通过抛 RuntimeException 异常来表示错误信息

Java 所有业务异常都应该是用 RuntimeException ，因为不用显示捕获或抛出，可以统一处理

Java 中如果函数返回值可能为 null，那么必须使用 Optional 来包装，提示调用者可能存在 null 值，Optional 除了函数返回值以外，不应该使用在其他任何地方 

Java 应用中，都应该定义一个顶级业务异常 BusinessException extends RuntimeException
