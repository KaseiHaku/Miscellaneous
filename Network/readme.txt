网络设备 Equipment:
    网卡(Interface): 
        运行位置: 物理层（电信号发送） + 数据链路层（链路建立和拆除，帧定界同步顺序差错控制等）
    
    中继器: 
        运行位置: 物理层
   

    集线器(HUB): 
        运行位置: 物理层
        n 个端口，任何端口接收数据，都广播到所有其他端口
        
    网桥(Bridge): 
        运行位置: 数据链路层
        一般只有两个端口，在两个端口间进行转发
        
        
        网桥是 硬件网络协议翻译器
        两个网络之间 协议不同，一个是 TCP/IP 一个是 令牌环，两个网络之间需要传输数据，就需要网桥进行翻译 转发。
        @attention 现在的操作系统之间为了互相交流，支持更多的协议，操作系统自己就可以是网桥，现在网桥这个概念已经淡出了
        @attention 网桥主要由软件实现，交换机主要由硬件实现!
    
    以太网(局域网)交换机(Switch): 
        运行位置: 数据链路层
        n 个端口，一个端口接收到数据，
        如果目标地址在 缓存中，则转发给 对应端口，否则 广播
        
        @attention 交换机使用 MAC 地址 和  ARP 协议
    
    路由器/三层交换机(Router)：
        运行位置: 网络层
        @attention 路由器使用 IP 地址 
     
    网关/四层交换机(Gateway)：
        运行位置: 传输层及以上
        可以将一个网络的 IP 数据包，发送给另一个网络的主机的设备，所以 网关 在各个网络中，都有一个属于该网络的 IP 地址
        @attention 即使
            比如有网络A和网络B，
            网络A的IP地址范围为“192.168.1.1~192. 168.1.254”，子网掩码为255.255.255.0；
            网络B的IP地址范围为“192.168.2.1~192.168.2.254”，子网掩码为255.255.255.0。
            在没有路由器的情况下，两个网络之间是不能进行TCP/IP通信的，
            即使是两个网络连接在同一台交换机(或集线器)上，TCP/IP协议也会根据子网掩码(255.255.255.0)判定两个网络中的主机处在不同的网络里。
            而要实现这两个网络之间的通信，则必须通过网关。
        

网卡：
    工作模式：
        广播模式（Broad Cast Model）：接收 IP packet 目标地址为 广播地址(0Xffffff) 的数据包
        多播模式（MultiCast Model）：接收 目标地址 为 多播地址 的数据包，无论是否在 多播组 内
        直接模式（Direct Model）：只接收目地址是自己 Mac 地址的帧
        混杂模式（Promiscuous Model）：网卡接收所有的流过网卡的帧，抓包程序就是在这种模式下运行的


AP(Wireless Access Point/无线访问接入点)：
    作用: 接收无线数据
    FIT AP(瘦 AP)： 也称 无线网桥、无线网关，相当于 有线网络中的集线器，本身无法进行配置，需要一台专门的 AC 进行集中控制管理配置
    FAT AP(胖 AP)： 也叫 无线路由器，一般具有 WAN LAN NAT DHCP DNS firewall VPN 等功能

AC(Wireless Acess Control/无线控制器)
    所用: 用于集中管理 Fit AP
    AC 一定是路由器，路由器不一定是 AC

################################ TP-Link 全屋覆盖 ################################ 
TP-Link 添加 有线子路由器            # 将 子路由器 当作 FIT AP 使用
    主路由器(直连光猫的那个)： 正常配置，保持不变
    副/子路由器:
    	上网设置
            基本设置：
    		    > WAN 口连接类型: 自动获得 IP 地址
    		WAN 口设置：
                > WAN口设定：固定 WAN 网口    （光猫出来的网线需要插到离 电源插孔 最近的一个 网口）
    		高级设置:
                >  数据包 MTU (字节): 1400
        无线设置: 正常配置即可
    	LAN 口设置
    		> LAN 口 IP 设置： 手动
    		> IP 地址: 192.168.0.2        # 一般 主路由 的地址为: 192.168.0.1
    		> 掩码: 255.255.255.0
    	IPv6 设置
            基本设置：
                > IPv6功能: 开启
    		    > WAN 口连接类型: 自动获取 IP 地址（DHCPv6/SLAAC）
    		高级设置：
                > IPv6 地址获取协议: DHCPv6
    		局域网设置: 
                > 主机配置: DHCPv6
    	DHCP 服务器:
    		> 关闭


TP-Link 中的 易展功能(WDS/Wireless Distibution System/无线分布式系统/无线桥接)：        # 上面有线方式的 无线版本
	主要功能： 通过 无线的方式 将多个 AP 连接在一起
	设置方式：
		关闭副路由器 DHCP 服务器
		开启副路由器 无线桥接 功能，并扫描 主路由器 无线信号
		点击桥接，输入 主路由器 密码，完成桥接
